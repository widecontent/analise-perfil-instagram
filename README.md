# 📊 Análise Gratuita de Perfil Instagram

Bem-vindo! Esta é uma landing page exclusiva para análise gratuita de perfil Instagram.

**Conta:** [@content.wide](https://instagram.com/content.wide)

## 🎯 O que é?

Uma ferramenta gratuita que oferece análise personalizada do seu perfil Instagram. A análise é exclusiva para quem segue @content.wide.

### ✨ O que você recebe na análise?

- Diagnóstico do seu perfil
- Identificação do que está funcionando
- 3 pontos principais de melhoria
- 1 ação urgente para implementar
- Recomendações personalizadas

## 🔄 Como funciona?

1. **Você acessa** o link da análise
2. **Você clica** em "Sim, já sigo" (validação)
3. **Você preenche** um formulário rápido (2 min)
4. **Você aguarda** a análise (até 5 dias úteis)
5. **Você recebe** no seu email + tem bônus no DM

## 📋 O que você precisa compartilhar?

- Seu nome completo
- Seu @ do Instagram
- Seu email
- Seu nicho/segmento
- Seu objetivo principal
- Seus maiores desafios
- Um screenshot dos seus últimos insights (ou descrição dos números)

## ⚙️ Setup (Configuração)

### Passo 1: Adicione o Formspree ID

Abra o arquivo `index.html` e procure por esta linha (aprox. linha 426):

```javascript
fetch("https://formspree.io/f/SEU_ID_AQUI", {
```

1. Va para [formspree.io](https://formspree.io)
2. Crie uma conta (grátis)
3. Crie um novo "form"
4. Copie o ID (tipo: `f/abc123xyz`)
5. Substitua `SEU_ID_AQUI` pelo ID

### Passo 2: Customize (opcional)

Se quiser mudar cores, textos, etc:
- Abra `index.html`
- Procure por `--cobalt`, `--cherry`, etc (no CSS)
- Mude os valores HEX
- Teste no navegador
- Faça commit e push
